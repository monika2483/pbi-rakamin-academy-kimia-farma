# pbi-rakamin-academy-kimia-farma

Project ini bertujuan untuk melakukan analisis dari data Kimia Farma periode 2020 - 2023. Adapun proses yang dilakukan dalam project ini yaitu :
1. Melakukan import dataset ke Big Query yang terdiri dari 4 table, yaitu table kf_final_transaction, table kf_inventory, table kf_kantor_cabang, table kf_product.
2. Melakukan proses query untuk menggabungkan keempat table dataset menjadi 1 table yang siap di analisis, dan diberi nama table_analisa.
3. Membangun dashboard visualisasi secara interaktif untuk mempermudah proses analisis data. Dashboarding dilakukan menggunakan Google Looker Studio.
4. Setelah dashboard selesai dibangun, langkah selanjutnya yaitu melakukan proses analisis untuk menemukan insight dari data serta dapat memberikan actionable recommendation.
